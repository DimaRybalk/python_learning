🎬 Lifeflix API

Це Backend-додаток для стрімінгового сервісу або бібліотеки фільмів, побудований на FastAPI та SQLAlchemy. Проєкт реалізує повноцінне управління користувачами, фільмами та жанрами з розмежуванням прав доступу.


🚀 - Основні можливості

Користувачі: Реєстрація, авторизація (JWT), перегляд профілю /me, видалення власного акаунта.

Адміністрування: Видалення користувачів, повне управління жанрами та фільмами.

Контент: Повний CRUD (створення, читання, оновлення, видалення) для фільмів та жанрів.

Обране: Можливість додавати фільми до списку "Моє обране" (зв'язок Many-to-Many).

Безпека: Хешування паролів (bcrypt), захист роутерів через залежності (Admin/User).

Валідація: Сувора перевірка даних через Pydantic (Email, довжина рядків, рік випуску).

Оптимізація: Пагінація списків та ефективне завантаження пов'язаних даних (selectinload).


🛠 Технологічний стек

Мова: Python 3.13

Фреймворк: FastAPI

ORM: SQLAlchemy 2.0 (Synchronous)

База даних: SQLite (файл database.db)

Валідація: Pydantic v2

Аутентифікація: PyJWT, Passlib (bcrypt)
