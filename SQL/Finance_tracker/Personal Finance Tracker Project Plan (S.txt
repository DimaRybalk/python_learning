Personal Finance Tracker Project Plan (SQLite + OOP + CRUD)
Project Goal

Build a console-based Personal Finance Tracker application using:
- Python (OOP principles)
- SQLite database
- CRUD operations
- Date handling
- Clean architecture (Models, Repositories, Services)

This project prepares you for FastAPI backend development.

Database Schema

Table: users
- id TEXT PRIMARY KEY
- email TEXT UNIQUE NOT NULL
- password TEXT NOT NULL
- created_at TEXT NOT NULL

Table: categories
- id TEXT PRIMARY KEY
- name TEXT NOT NULL
- type TEXT CHECK(type IN ('income','expense')) NOT NULL

Table: transactions
- id TEXT PRIMARY KEY
- user_id TEXT NOT NULL
- category_id TEXT NOT NULL
- amount REAL NOT NULL CHECK(amount > 0)
- date TEXT NOT NULL
- description TEXT
- created_at TEXT NOT NULL
- FOREIGN KEY(user_id) REFERENCES users(id)
- FOREIGN KEY(category_id) REFERENCES categories(id)
